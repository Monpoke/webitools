# WebiTools 
C'est en août 2012 que ce projet fut créé. Son but était de fournir des fonctionnalités supplémentaires pour tous les sites créés avec [Webidev](http://webidev.com). Le service est toujours hébérgé à l'adresse http://webis.plumedor.fr. 

Ces sources sont hébergées à des fins historiques.

Bonne découverte !
